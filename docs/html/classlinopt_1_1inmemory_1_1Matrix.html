Detailed Description

template<typename E = double>
class linopt::inmemory::Matrix< E >

The Matrix<E> class stores nxm entries of type E.

This class handles dense matrices. The type E must model a commutative ring, that is: for e1, e2 of type E, e1 * e2, e1+e2 must be well defined and of type E.
